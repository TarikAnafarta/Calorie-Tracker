# Calorie Tracker

A modern Android application for tracking daily calorie intake, managing nutrition goals, and monitoring meal history.

## Features

* Personalized calorie goals based on BMR calculations
* Hybrid food search using a local database and OpenFoodFacts API
* Meal tracking with customizable portion sizes
* Daily calorie progress dashboard
* Persistent meal history with Room Database
* Dark mode support

## Tech Stack

* Kotlin
* Jetpack Compose
* Room Database
* DataStore
* OpenFoodFacts API
* MVVM Architecture

## Overview

Calorie Tracker simplifies nutrition management by combining fast local food search with real-time nutritional data from external sources. Users can track meals, monitor calorie consumption, and maintain a history of their dietary habits through a clean and intuitive interface.

## Future Improvements
